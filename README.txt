LankaDrive Auto Sales - HTML/CSS Website

Files included:

- about.html
- vehicles.html
- services.html

- css/style.css

1. Extract the ZIP.
2. Open index.html in a browser.
3. Keep the folder structure unchanged.

Note:
The image files included are stylized placeholder assets so the site works fully offline.
You can replace them with real car photos later without changing the HTML structure.
